USELESS WEB

This workshop will attempt to introduce the basics of the web (html + css) through the use of templates, tampering, and copy-pasting with the goal of producing a website that evades the typical uses and values that are deemed "useful" in the web today. We can think of the web as a fast, flowing, military-industrial rapids lined with pockets of calm eddies.


(let's head for the eddy)

What else can we use the web for besides providing endless nourishment to the imperial platforms that we claim as home today?

Questions to ask when making a website:

  1. who might my website harm?
  2. who is my website for?
  3. what data can be collected from my website?
  4. is my website useful to the military?

Consider: website as

  1. poem
  2. still life
  3. self-portrait
  4. humidifier

Where to begin?

We'll need two essential tools: a browser and a text-editor.
The browser you almost certainly already have--but for our sake we'll use either Chrome or Firefox.
For text-editors, you can use any simple text-editor like notepad but I recommend Atom, Sublime, or Brackets.
